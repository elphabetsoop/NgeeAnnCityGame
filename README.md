# NgeeAnnCityGame
City-building web game for SPM Assignment 2

## Build
```
npm install
npm start
```

## Contributing
**File structure:**
```
├── *.html
├── assets/ # images
├── css/
|    └─ styles.css
|    └─ leaderboard.css
└─── scripts/
     └─ arcade.js
     └─ freeplay.js
     └─ leaderboard.js
     └─ load-save-game.js
     └─ scoring.js
     └─ *_tutorial.js
```
